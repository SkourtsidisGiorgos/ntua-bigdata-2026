# Δήλωση χρήσης εργαλείων LLM

Εργαλεία που χρησιμοποιήθηκαν:

- Claude (Anthropic, μοντέλα Opus 4.8 & Sonnet 4.6)
- GLM (z.ai, μοντέλο GLM 5.2)
- DeepSeek (μοντέλο v4-pro)

| Τμήμα                                                                                                                 | Έκταση χρήσης                                                           | Τι έγινε χωρίς LLM                                                                                                                    |
| --------------------------------------- | ------------------------- | --------------------------------------------- |
| Στήσιμο περιβάλλοντος (Spark/Hadoop clients, VPN, k8s)                                                                | Παρήγαγε τα scripts (`install_clients.sh`, `vpn_up.sh`, `env.sh`)       | Επιλογή cluster, εκτέλεση /login, διάγνωση του προβλήματος DNS του VPN                                                                |
| Κοινός κώδικας (`common/`: `Datasets`, `Paths`, `TimeBucket`, `Geo`, `JoinStrategy`, `Bench`, `Output`) + Queries 1-4 | Χρήση συμβουλευτικά + review + σχόλια στον κώδικα                       | Σχεδίαση (loaders, normalize columns), επιλογή Java 17, έλεγχος ορθότητας δεδομένων, εκτέλεση, έλεγχος αποτελεσμάτων/χρόνων, ερμηνεία |
| Scripts εκτέλεσης (`submit.sh`, `put_hdfs.sh`, `logs.sh`, `test.sh`)                                                  | Γράψιμο από LLM με συγκεκριμένη καθοδήγηση                              | Καθορισμός απαιτήσεων, εκτέλεση στο cluster                                                                                           |
| Μετατροπή format (CSV -> Parquet)                                                                                     | Βοήθησε στο development του converter και του benchmark                 | Επιλογή Parquet, ερμηνεία αποτελεσμάτων                                                                                               |
| Τοπικά JUnit tests                                                                                                    | Ρυθμιση του surefire plugin                                             | Γραψιμο των τεστς                                                                                                                     |
| Notebook EDA (`notebooks/eda.ipynb`)                                                                                  | Σκελετός από LLM                                                        | Καθορισμός της ανάλυσης, συμπεράσματα                                                                                                 |
| Πίνακες αναφοράς                                                                                                      | Συνέταξε τους πίνακες σε LaTeX από τα πραγματικά μετρημένα αποτελέσματα | Εκτέλεση όλων των μετρήσεων στο cluster, συγγραφή αναφοράς                                                                            |
| Aναφορά                                                                                                               | Boήθησε με διάφορα bugs του LaTeX αρχείου (μορφοποίηση)                 | Συγγραφή αναφοράς                                                                                                                     |
